# 系统{#system}
## 什么是系统？
系统是计算机的硬件和软件集合，它提供了计算机的硬件和软件功能。   
## 系统的组成
硬件：计算机的物理结构，如CPU、内存、硬盘、显示器、键盘等。  
软件：计算机的逻辑结构，如操作系统、应用程序、数据库、网络等。  
硬件和软件是系统组成。  
系统是计算机的硬件和软件集合，它提供了计算机的硬件和软件功能。    
## 系统的分类
1. 单用户系统：一个用户只能使用一个系统。
2. 多用户系统：多个用户可以同时使用一个系统。
3. 分布式系统：多个用户可以同时使用多个系统。
4. 集群系统：多个用户可以同时使用多个系统，但是每个系统只能有一个用户使用。
5. 虚拟系统：一个用户可以同时使用多个系统。
6. 容器系统：一个用户可以同时使用多个系统，但是每个系统只能有一个用户使用。
7. 虚拟化系统：一个用户可以同时使用多个系统，但是每个系统只能有一个用户使用。
